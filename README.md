# Sarthak Gambhir — Developer Portfolio

A personal portfolio website showcasing my work, skills, and contact information. Built as a static site with HTML and CSS—no build step or dependencies required.

## Features

- **Hero** — Introduction and call-to-action links
- **About** — Background and internship goals
- **Skills** — Frontend, backend, and language stack
- **Projects** — Highlighted work (e.g. Online Voting System)
- **Contact** — Email, LinkedIn, GitHub, and location
- Responsive layout with smooth scrolling navigation

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid)
- [DM Sans](https://fonts.google.com/specimen/DM+Sans) & [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) via Google Fonts

## Project Structure

```
portfolio-website/
├── index.html    # Page content and sections
├── style.css     # Layout, typography, and theme
└── README.md
```

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/sarthakgambhir/portfolio-website.git
   cd portfolio-website
   ```

2. Open `index.html` in your browser, or serve the folder locally:

   ```bash
   # Python 3
   python -m http.server 8000
   ```

   Then visit [http://localhost:8000](http://localhost:8000).

## Customization

Edit `index.html` for content (projects, skills, links) and `style.css` for colors and layout. Theme variables are defined at the top of `style.css` (`--black`, `--accent`, `--cream`, etc.).
